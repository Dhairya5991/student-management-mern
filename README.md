# 🎓 Student Management System (MERN Stack)

A full-stack **Student Management System** built using the **MERN Stack (MongoDB, Express.js, React.js, Node.js)**.  
This application allows users to efficiently manage student records with a modern UI and RESTful API.

---

## 🚀 Features

- ➕ Add new students
- 📋 View all student records
- ❌ Delete student records
- 🌐 Responsive UI (Frontend with React)
- 🔗 REST API (Backend with Node.js & Express)

> These features are commonly found in MERN-based student systems with CRUD operations and search capabilities. :contentReference[oaicite:1]{index=1}
